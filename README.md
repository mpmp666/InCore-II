## 🎯 项目定位

InCore II 是 InCore 的全面重构与高强度补充版本。核心代码完全自主研发，深度优化底层架构，致力于成为 **MCPE 0.14 时代超越 OBS 的主流服务端核心**，为玩家提供极致的原版生存体验。

## ✨ 核心亮点

### 🔴 完整红石系统
- ✅ 红石线、红石火把、中继器、比较器、红石灯、门、活板门、漏斗、发射器、投掷器、音符盒等全组件支持
- ⚡ 高频红石脉冲与同 tick 更新处理，杜绝卡顿与延迟异常
- 🧭 中继器/比较器按玩家视角精准放置，方向逻辑100%还原原版

### 🌾 真实农田机制
- 👣 踩踏退化：玩家/实体高处坠落触发耕地退化
- 💧 缺水干涸：耕地逐步失水，严重时恢复为泥土
- 🌱 作物保护：退化时自动顶掉上方作物，行为与原版完全一致

### 🌍 沉浸式地形生成
- 🌿 地表植被与群落结构更加丰富自然
- 🌻 支持两格高植物（大型草丛、双高植物等）
- 📐 植被分布规律有序，告别稀疏杂乱
